A full stack developer is someone who can develop both the client and server software. Some of the tools used by a full stack developer includes but nit limtted to Git/Github, Gitbash, text editor (e.g Vscode, Sublime text editor, Atom etc). The following are the various programming laguages deployed by a full stack developer; HTML, CSS, Javascript, jQuery, Angular, React or Vue) for the client side of the software and (PHP, ASP, Python, Java, or Node) for the server side.
